# Usage

Comprehensive guides for all ytdl-go features.

This section covers:

- **Format Selection** - Choose video and audio formats
- **Output Templates** - Organize your downloads
- **Metadata and Sidecars** - Handle video metadata
- **Basic Downloads** - Download videos with various options
- **Audio-Only Downloads** - Extract audio from videos
- **Playlist Downloads** - Download entire playlists
