# Tech Stack

## Backend
- **Language:** Go (1.24+)
- **YouTube Interaction:** `github.com/kkdai/youtube/v2`
- **Bot Authentication:** `https://github.com/LuanRT/BgUtils` (Integrated for PO Token and bot attestation)
- **Terminal UI (TUI):** `github.com/charmbracelet/bubbletea`
- **Media Processing:** `github.com/u2takey/ffmpeg-go` (ffmpeg wrapper)
- **Metadata Management:** `github.com/bogem/id3v2/v2`

## Frontend
- **Framework:** SolidJS
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Icons:** `lucide-solid`

## Development & Build
- **Orchestration:** Custom `build.sh` script for unified backend and frontend build/launch processes.
