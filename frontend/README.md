# Frontend

This directory contains the SolidJS single-page application (SPA) for the `ytdl-go` project.

## Subdirectories

*   `src`: Contains the source code for the SolidJS application.
*   `docs`: Contains documentation specific to the frontend.

## Development

To get started with frontend development, run the following commands:

```bash
cd frontend
npm install
npm run dev
```
