# Internal

This directory contains the private application and library code for the `ytdl-go` project. It is not meant to be imported by other projects.

## Subdirectories

*   `app`: Contains the main application logic that orchestrates the different parts of the application.
*   `downloader`: Contains the core logic for downloading videos from YouTube and other websites.
*   `web`: Contains the web server and API for the SolidJS frontend.
*   `docs`: Contains documentation specific to the internal workings of the project.
