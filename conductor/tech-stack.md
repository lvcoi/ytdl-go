# Core Stack
- **Backend Language:** Go 1.24+ (Leveraging high concurrency and performance)
- **Frontend Language:** JavaScript (SolidJS/JSX)
- **Backend Frameworks:** 
    - [Bubble Tea](https://github.com/charmbracelet/bubbletea) (TUI Engine)
    - [Goja](https://github.com/dop251/goja) (ECMAScript runtime for PO token handling)
    - [ytdl-lib](https://github.com/lvcoi/ytdl-lib) (Core YouTube extraction library)
- **Frontend Frameworks:** 
    - [SolidJS](https://www.solidjs.com/) (Reactive UI library)
    - [Vite](https://vitejs.dev/) (Build tool and dev server)
    - [Tailwind CSS](https://tailwindcss.com/) (Utility-first styling)

# Integration & Infrastructure
- **API:** REST API serving the SolidJS Single Page Application (SPA)
- **Build System:** Integrated `build.sh` script orchestrating Go and NPM builds
- **Version Control:** Git
